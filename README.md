# Simpsons Random Characters Collection

This project is a small React application built with Vite that lets users discover random characters from The Simpsons universe. Each character card displays key information such as name, ID, status, age, birthdate, gender, occupation, and a random quote.

- Show a random character card
- Save selected characters to a personal collection
- Display character details in a styled UI
- Smooth scrolling when a new character appears
